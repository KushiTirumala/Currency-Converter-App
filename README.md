Currency Converter App

A simple and interactive Currency Converter built using HTML, CSS, and JavaScript. Converts amounts between different currencies in real-time with a user-friendly interface.

Project Overview

This project demonstrates frontend development skills by creating a web-based currency converter. Users can select currencies, input amounts, and get instant conversion results using JavaScript logic.

Features

Convert amounts between multiple currencies

Real-time calculation using JavaScript

User-friendly and responsive interface

Easy to use and customize

Technologies Used

HTML5 – Structure of the app

CSS3 – Styling and layout

JavaScript – Conversion logic and interactivity
